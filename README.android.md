# 闪耀星骑士 Android 动画替换 MOD

## 暂停提供

安卓版 Release 已暂时撤下，请不要继续使用先前发布的 Android 安装器。

经核查，先前安装器只兼容开发期间使用的自制测试 APK。文档所指向的 `anosu/DMM-Mod` 公开 APK 使用内置 Frida 脚本，不会加载安装器推送到手机的外部脚本，因此会提示：

```text
The installed app is not a compatible Android package (APK)
```

标准版和 legacy 版公开 APK 均采用相同加载方式，切换版本、USB 模式或重新运行 BAT 都无法解决这个兼容问题。

现有安卓版源码会保留，待完成兼容公开 APK 的安装方案并通过实际设备验证后再重新发布。PC 版不受影响。
