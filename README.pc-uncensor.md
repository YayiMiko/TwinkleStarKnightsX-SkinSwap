# 闪耀星骑士 PC 去码验证版

这是独立维护的 PC 去码验证版 0.1。它保留通常版的“通常攻击 2”动画替换功能，并移除变身动画中能够识别的马赛克局部。“通常攻击 1”仍使用游戏原始演出，实际攻击、伤害和战斗速度不会改变。

## 使用前请注意

游戏资源通常没有藏在马赛克下面的完整无码原画。去掉马赛克后，多数角色只会显示平滑肤色、简单阴影或留白，通常看不到更多身体细节。这是资源本身的结果，不是安装失败。

本版不会重绘画面，也不能识别所有角色。个别角色可能仍有马赛克，或因误判附件而出现身体局部、头发或特效缺失。露露（`[炎宿せし宝石] ルルゥ`）仍不应用 MOD。

## 安装

1. 完全关闭游戏并完整解压下载包。
2. 将其中的 `TskSkinSwap` 文件夹覆盖到 `<游戏目录>\mods\TskSkinSwap\`。
3. 双击 `TskSkinSwap` 文件夹里的 `Apply-TskSkinSwap.bat`。
4. 显示 `Completed successfully` 后，经由 DMM 启动游戏。

安装工具会复用已经下载且仍然有效的动画资源。

## 建议验证内容

- “通常攻击 1”保持原始演出。
- “通常攻击 2”正常替换，且马赛克局部被移除。
- 人体、头发、衣物和特效没有异常缺失。
- 连续攻击、切换角色、图鉴后进入战斗及重启游戏均不白屏或卡死。

出现异常时，请保留 `<游戏目录>\BepInEx\config\TskSkinSwap\runtime.log` 并报告角色。

## 恢复通常版

完全关闭游戏，用 PC 1.3.0 通常版覆盖 `mods\TskSkinSwap`，再运行其中的 `Apply-TskSkinSwap.bat`。已经下载的有效动画资源可以继续复用。
