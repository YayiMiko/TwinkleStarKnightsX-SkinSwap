# 更新记录

## Android 版暂停提供 - 2026-07-13

- 确认公开 APK 不会加载本项目安装的外部脚本，先前安卓版安装器与公开 APK 不兼容。
- 撤下 Android Release 和下载入口，等待兼容方案完成并通过设备验证后重新发布。

## PC Release 1.1.2 - 2026-07-13

- 修复在图鉴或战斗中播放过替换动画后，进入后续战斗时角色可能消失、只剩通常攻击通用特效的问题。
- 场景切换产生新的通常攻击骨骼实例时，会自动重新准备变身资源。

## PC Release 1.1.1 / Android Release 0.1.2 - 2026-07-13

- PC 与 Android 均暂时跳过露露（`[炎宿せし宝石] ルルゥ`，`1141001`），保留该角色的游戏原始演出。
- 下载器不再获取露露的变身资源，已有旧映射也会由运行组件强制跳过。
- 安卓版会在用户未完整解压 ZIP 时直接提示缺少文件，不再显示难懂的 PowerShell 路径错误。

## Android Release 0.1.1 - 2026-07-13

- 修复部分中文 Windows 环境双击安装或卸载 BAT 后出现“不是内部或外部命令”的问题。
- 补充手机 USB 调试授权、电脑安装结果和游戏自动启动的完整确认流程。

## PC Release 1.1 - 2026-07-13

- 修复安装 MOD 后在图鉴回放抽卡动画时可能白屏或卡死的问题。
- 支持战斗和图鉴预览中“通常攻击 1”和“通常攻击 2”的成人变身动画替换。
- 改为按实际使用情况准备动画资源，减少无关资源占用。
- 支持从 PC 版 1.0 直接覆盖升级，无需先卸载，已有有效下载会继续复用。

## Android Release 0.1 - 2026-07-12

- 支持战斗和非战斗预览中“通常攻击 1”和“通常攻击 2”的成人变身动画替换。
- 自动读取当前游戏资源目录，并优先复用手机已有缓存。
- 提供一键安装、更新和卸载，无需 Root。
- 修复主页异常和原变身动画白屏问题。

已知限制：部分超宽屏手机在动画缩放到最小时可能露出两侧战斗背景。

## PC Release 1.0 - 2026-07-12

- 支持将“通常攻击 1”和“通常攻击 2”的角色演出替换为对应的高画质成人变身动画。
- 自动匹配支持角色，无需逐个打开变身界面准备缓存。
- 提供一键安装、游戏更新后重新应用和卸载功能。
- 下载中断后可继续运行，已完成资源会被复用。
