# 闪耀星骑士 Android 动画替换 MOD

## Android Release 0.3.1

Android 0.3.1 改进了首次安装和游戏更新后的操作流程：需要操作手机时会显示醒目的中文提示，并等待用户开启 USB 调试、解锁手机和确认最终安装。新版还修复了中文 Windows 路径导致安装失败，以及安装结束后解压文件夹可能被 ADB 占用的问题。它同时包含 0.3.0 的动画稳定性修复。请下载 [Android Release 0.3.1](https://github.com/YayiMiko/TSKSkinSwap/releases/tag/android-v0.3.1)，不要继续使用先前的 Android 安装器。

新方案会由电脑自动完成以下操作：

1. 每次运行时查询 `anosu/DMM-Mod` 的 GitHub Release，获取最新的标准兼容安卓安装包，并核对 GitHub 公布的文件大小和 SHA-256。
2. 将本 MOD 与原有汉化脚本一起内置，保留汉化功能。
3. 通过电脑覆盖安装，不卸载游戏，也不清除手机上的大型游戏数据。
4. 优先复用游戏缓存和 MOD 已下载资源，只补齐缺失的高画质变身资源。
5. 写入最新角色映射并自动拉起手机上的游戏。

已确认通常攻击 1、通常攻击 2、原生变身动画和连续切换角色均可正常使用。新版让通常攻击与游戏原生变身动画共用游戏自身的资源加载流程，不再重复打开同一份动画资源。

首次请求某个角色时，如果资源仍在加载，本次通常攻击会保留游戏原演出。等待片刻或先播放一次该角色的原生变身动画，再次播放通常攻击即可正常替换；这不会造成永久失效。

露露（`[炎宿せし宝石] ルルゥ`）仍不支持，游戏会保留该角色的原始通常攻击演出。

## 使用流程

1. 手机需已安装 [anosu/DMM-Mod](https://github.com/anosu/DMM-Mod) 提供的标准兼容安卓安装包。先正常启动一次游戏，完成游戏更新和大型数据下载，再关闭游戏。
2. 在手机“关于手机”中连续点击“软件版本”或“版本号”约 7 次，按提示开启开发者模式；然后在“开发者选项”中打开“USB 调试”。不同品牌的菜单名称可能略有差异，可在设置中直接搜索“开发者选项”。
3. 用数据线连接电脑。手机出现“允许 USB 调试”时选择允许；若出现“允许通过 USB 安装”，也需要在手机上确认。
4. 在电脑上完整解压发布包，双击 `Apply-TskSkinSwap-Android.bat`。首次运行会下载所需工具、友仓最新的兼容安装包和缺失的变身资源，请勿中途断开手机。
5. 如果脚本发现友仓安装包比手机上的版本新，会先更新应用并启动游戏。此时请在手机游戏内完成更新，然后关闭游戏，再次运行同一个 BAT。
6. 电脑显示 `Installed ... mappings` 后，会出现“现在请操作手机”的中文提示。请解锁手机并保持亮屏；如果手机询问是否允许通过 USB 安装，请选择允许，然后回到电脑按 Enter 继续。
7. 电脑显示绿色的“MOD 安装完成”，并且手机上的游戏被自动拉起后，才表示安装完成。只有兼容应用更新成功、但尚未再次运行 BAT，不算完成。

如果电脑显示正在等待 USB 调试授权，请解锁手机，在“允许 USB 调试？”窗口中勾选“始终允许使用这台计算机”并点“允许”。新版安装器会等待 90 秒，不需要立刻重开 BAT。手机没有弹窗时，可先关闭再开启“USB 调试”，拔插数据线后重试；同时确认使用的是支持数据传输的线，而不是只能充电的线。

Windows 用户名或解压路径可以使用中文；安装器会自动把不支持中文路径的临时处理步骤放到纯英文目录中。

游戏更新后，旧配置会自动停用。重新运行 BAT；如果脚本先更新了兼容安装包，请在手机游戏内完成更新，关闭游戏后再运行一次 BAT。有效的游戏缓存、MOD 资源和电脑下载会继续复用；下载中断时也可以直接重新运行。

需要恢复时，完整解压当前发布包，连接并授权手机后运行 `Uninstall-TskSkinSwap-Android.bat`。卸载工具可以独立准备恢复所需文件，不要求先在同一文件夹运行安装器；首次运行可能需要下载约 250 MB。恢复完成时手机上的游戏也会自动启动。ADB 会从 Windows 临时目录运行，不再占用发布包文件夹；关闭命令行窗口后即可删除解压文件夹。默认保留已下载变身资源，方便以后重新安装。

## 无需电脑版本

目前 Android 版仍需 Windows 电脑、数据线和 USB 调试。无需电脑、直接在安卓手机上完成安装与更新的版本正在开发中。
