# 闪耀星骑士 Android 动画替换 MOD

## Android Release 0.2.6

Android 0.2.6 已完成完整真机验证。请下载 [Android Release 0.2.6](https://github.com/YayiMiko/TSKSkinSwap/releases/tag/android-v0.2.6)，不要继续使用先前的 Android 安装器。

新方案会由电脑自动完成以下操作：

1. 从 `anosu/DMM-Mod` 的 GitHub Release 获取本 MOD 已验证的标准兼容安卓安装包，并核对完整文件的 SHA-256。
2. 将本 MOD 与原有汉化脚本一起内置，保留汉化功能。
3. 通过 ADB 覆盖安装，不卸载游戏，也不清除手机上的大型游戏数据。
4. 优先复用游戏缓存和 MOD 已下载资源，只补齐缺失的高画质变身资源。
5. 写入最新角色映射并自动拉起手机上的游戏。

已在 Android 16 真机确认：首页和汉化正常；通常攻击 1、通常攻击 2、连续击杀、图鉴预览后进入战斗、场景切换及完整重启均正常。新版还会核对游戏版本和资源目录，避免游戏更新后继续加载过期配置。

露露（`[炎宿せし宝石] ルルゥ`，内部 ID `1141001`）仍不支持，游戏会保留该角色的原始通常攻击演出。

## 使用流程

1. 手机需已安装 [anosu/DMM-Mod](https://github.com/anosu/DMM-Mod) 提供的标准兼容安卓安装包。先正常启动一次游戏，完成游戏更新和大型数据下载，再关闭游戏。
2. 在手机“关于手机”中连续点击“软件版本”或“版本号”约 7 次，按提示开启开发者模式；然后在“开发者选项”中打开“USB 调试”。不同品牌的菜单名称可能略有差异，可在设置中直接搜索“开发者选项”。
3. 用数据线连接电脑。手机出现“允许 USB 调试”时选择允许；若出现“允许通过 USB 安装”，也需要在手机上确认。
4. 在电脑上完整解压发布包，双击 `Apply-TskSkinSwap-Android.bat`。首次运行会下载所需工具、本 MOD 已验证的标准兼容安卓安装包和缺失的变身资源，请勿中途断开手机。
5. 电脑显示 `Installed ... mappings`，并且手机上的游戏被自动拉起后，才表示全部完成。只有电脑提示 APK 安装成功、但手机游戏没有启动，不算完成。

游戏更新后，旧配置会自动停用。请先启动游戏完成更新，再关闭游戏并重新运行 BAT。有效的游戏缓存、MOD 资源和电脑下载会继续复用；下载中断时也可以直接重新运行。

需要恢复时，完整解压当前发布包，连接并授权手机后运行 `Uninstall-TskSkinSwap-Android.bat`。卸载工具可以独立准备 ADB 和恢复用的标准兼容安卓安装包，不要求先在同一文件夹运行安装器；首次运行可能需要下载约 250 MB。工具下载使用多个 Google 官方地址，连接失败或超时会自动切换，下载结果仍会核对固定 SHA-256。恢复完成时手机上的游戏也会自动启动。安装或卸载流程结束后会关闭随包启动的 ADB 服务，关闭命令行窗口后即可删除解压文件夹。若资源传输期间 ADB 服务意外退出，安装器会自动重启并重试一次。默认保留已下载变身资源，方便以后重新安装。
